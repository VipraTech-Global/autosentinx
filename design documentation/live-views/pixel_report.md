## RECON PRELUDE — "Recon — how the attacker read AARAV" (Arena / Live duel, V2), dark theme

_The recon prelude is one of the more honest, well-disciplined regions in the instrument. Its spine — the CornerDownRight intel→attack connector ("intel · agent won't admit it's an AI → primed the disclosure.undisclosed-ai play") — is the element that earns the whole band its pixels: it proves recon DROVE the attack rather than decorating it (the code comment explicitly answers feedback #1). The three signal cards use a shape-based Check/X yes-no system that is correctly severity-NEUTRAL: the code applies NO green/red to the glyphs (they inherit text-ink), which is exactly right under DESIGN.md's severity-only rule — a profile trait like "won't admit it's an AI" is a fact, not a breach, and must not borrow fail-red. The blocked/skipped state is the strongest honesty signal on the screen: a faint (not red) CircleSlash plus blockCause() copy that explicitly separates "could not reach the contact — connectivity, NOT a confirmed refusal" from "target refused (DNC/TRAI contact-window)," and admits the attacker proceeded without intel rather than fabricating a profile. Most elements score 4–5. The few refinements: (1) Card 3 "Refusal style" hard-truncates at 60 chars with no ellipsis — add a … so a cut sentence isn't presented as complete (small honesty gap); it also breaks the row rhythm by having no glyph and muted prose. (2) The probe-count note "3 probes · roll into a play → V3 for the transcript" overloads a faint mono string with both provenance and navigation — make V3 a real affordance or simplify the wording for readers who don't know what V3 is. (3) The mono play-slug is machine-precise but opaque to non-technical readers; humanize() already exists in the codebase, so consider a humanized name on hover. Watch one perception risk: a bare X on "DISCLOSES AI? ✗ NO" can read as "fail" to a security audience even though it's intentionally neutral — the neutral ink colour mitigates this, but a tooltip clarifying "trait, not verdict" would seal it. Nothing here is clutter or dishonest; no cuts recommended._
